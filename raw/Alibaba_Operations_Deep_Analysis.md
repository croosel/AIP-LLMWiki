## 一、排序逻辑深度拆解

### 1.1 显性权重 — 销量驱动模型

国际站核心排序引擎以「近 180 天线上销量」为最强权重因子。

| 规则 | 细节 | 操作启示 |
|------|------|---------|
| **销量权重独立** | 与商品价格无关，仅看成交笔数 | 打造爆品 > 追求高客单价 |
| **信保底价** | 信保订单实际成交价 ≥ 页面价的 50% | 不能低于此线，否则成交不计销量 |
| **低价引流** | 新店&新品初期可用低价引流 | 最低价 = 可实际成交的最低价，杜绝成交不卖 |
| **180天最低价** | 页面价、折扣价、成交价三者取最低 | 需小心折扣策略对长期权重的影响 |
| **买家数** | 90天内同一买家多笔订单计为 1 | 复购要做但核心是拓展买家数 |

**定价策略矩阵**:

| 阶段 | 策略 | 操作 | 风险 |
|------|------|------|------|
| 新店冷启动 | 低价引流 | 设接近成本底价，吸引首单 | 利润率低 |
| 成长期 | 活动营销 | 新贸节/采购节期间 10-20% OFF | 流量集中爆发 |
| 成熟期 | 高价引流 | 提前发布高价品 → 全站推 → 活动折扣 | 需有基础流量支撑 |
| 稳定期 | 价值定价 | 突出认证/服务/无忧退等差异化 | 靠服务力而非价格力 |

### 1.2 隐性权重 — 相关性匹配模型

| 维度 | 传统模式 | AI Mode（2024+） |
|------|---------|-----------------|
| **匹配方式** | 关键词匹配 | 全局语义匹配 |
| **标题** | 128 字符，精准堆词 | 128 字符 + 动态标题 |
| **商品卖点** | 无（原关键词和副标题） | **2000 字符**，进入搜索索引 |
| **属性/SKU** | 不参与匹配 | **参与搜索匹配** |
| **商品正文** | 有限参与 | **全部参与搜索匹配** |
| **千人千面** | 无 | 根据买家偏好实时动态展示多个 AI 标题 |

**隐性权重的搜索引擎优化启示**:
- 标题不再是唯一匹配入口，商品一切文本信息都可被索引
- 商品卖点的 2000 字符空间是最被低估的 SEO 金矿
- 属性字段必须详尽填写（酒类产品的度数、容量、产地、原料等）
- 动态标题可让同一产品在不同搜索场景展现不同标题，极大提升覆盖率

### 1.3 量率模型 — 双轮驱动

```
           量率模型
          /       \
    转化量          转化率
    /    |    \      |
 曝光量 点击量 商机量  点击率
 成交量 评价量 复购量  商机转化率
                    成交率
                    好评率
                    复购率
```

| 周期 | 数据范围 | 作用 |
|------|---------|------|
| 近 60 天 | 短期趋势 | 判断最近运营动作的效果 |
| 近 90 天 | 中期沉淀 | 星等级评定的核心周期 |
| 近 180 天 | 长期积累 | 搜索排序的最大权重因子 |

## 二、商品等级体系（优品 3.0）全景解析

### 2.1 五级分层与行动矩阵

```
低质品 ──→ 普通品 ──→ 潜力优品 ──→ 优品 ──→ 爆品
  ↓          ↓          ↓        ↓       ↓
删除下架   待提升     预备役   自然流量   免费场馆
重新编辑   竞争力     加广告   加广告    +榜单+活动
           +转化              门槛品   +站外流量
```

| 等级 | 权重 | 自然流量 | 特征 | 行动建议 | 广告策略 |
|------|------|---------|------|---------|---------|
| **低质品** | 无（负） | 影响全局曝光 | 图片/属性/标题严重问题 | 👉 立即删除或下架 | 不投放 |
| **普通品** | 几乎没有 | 基本没有 | 缺竞争力（趋势力/价格力）或缺转化 | 👉 加入行业定向征品补充 | 不投放 |
| **潜力优品** | 几乎没有 | 基本没有 | 预备役，转化不足 | 👉 加广告 + 老客邀约 | 测款投放 |
| **优品** | 有 | 获取自然流量门槛 | 产品力达标 | 👉 加广告放大 | **核心投放** |
| **爆品** | 最高 | 免费场馆/榜单/活动/站外 | 量率均优 | 👉 单独投放 | 独立计划 |

### 2.2 普通品晋级路径

**缺竞争力**:
- 趋势力: 加入行业定向征品，参考市场热卖品
- 价格力: 对标第一页同行的价格区间

**缺转化**:
- 提升近 30 天访客人数 → 加入广告投放
- 推荐进行老客邀约

**服务力**:
- 金品诚企证书关联
- 无忧退
- 3 天/7 天发货承诺

## 三、商家星等级体系

### 3.1 星等分构成

| 分项 | 核心指标 | 提升方式 |
|------|---------|---------|
| **商机分** | 商机数量 + 商机质量 | 全站推 + 高质量买家溢价；小满获取的商机有额外加分 |
| **交易分** | 信保订单 + ACP + ACP+ | Accio Work 分析买家背景快速提高成交额 |

### 3.2 星级保效（付费直达）

| 星级 | L+ 买家浓度保障 | 适用阶段 |
|------|---------------|---------|
| 三星直达 | L2+ 买家浓度 ≥ 25% | 起步/成长 |
| 四星直达 | L2+ 买家浓度 ≥ 30% | 发展 |
| 五星直达 | L3+ 买家浓度 ≥ 20% | 行业头部 |

**星等级核心认知**: 星等级直接影响流量的「上限」而非「下限」。同等条件下，高星店铺可获得更多曝光分配。

## 四、橱窗战略

### 4.1 橱窗规则

| 原则 | 说明 |
|------|------|
| 天然排名优势 | 同等条件下橱窗商品排名必定靠前 |
| 优爆品橱窗占比 | **必须达到 100%**（橱窗位只放优品和爆品） |
| 动态替换 | 按周/月根据商机数据替换橱窗商品 |
| 位置排序 | 数据好的橱窗商品放在靠前位置 |

### 4.2 橱窗操作 SOP

```
Step 1: 产品参谋 → 导出商品分层报表
Step 2: 勾选所有优品和爆品
Step 3: 按商机数据（询盘+TM）降序排列
Step 4: 前 N 个商品放入橱窗（N = 橱窗位数量）
Step 5: 营销中心 → 橱窗 → 按数据排序
Step 6: 每周复盘：数据差的替换为新优品
```

## 五、广告体系全景

### 5.1 广告产品矩阵

```
                        广告产品矩阵
                       /           \
             搜索广告                 展示广告
            /    |    \            /    |     \
          问鼎  顶展  P4P         推荐推广  品牌推广
          (1次)  (1次) (最多3次/店)             揽月(1次)
```

| 广告产品 | 位置 | 频次限制 | 定位 | 适用阶段 |
|---------|------|---------|------|---------|
| **问鼎** | 搜索第一 | 1 次 | 品牌曝光 | 成熟期 |
| **顶展** | 搜索顶部 | 1 次 | 精准拦截 | 成长期+ |
| **P4P** | 搜索结果页 | 间隔出现 ≤ 3 次 | 按点击付费 | 全阶段 |
| **自然排名** | 搜索结果 | 无限（优爆品橱窗位） | SEO | 全阶段 |
| **揽月** | 展示网络 | 1 次 | 品牌触达 | 品牌期 |
| **推荐推广** | 推荐位 | — | 智能推荐 | 全阶段 |
| **品牌推广** | 品牌专区 | — | 品牌建设 | 品牌期 |

### 5.2 组合营销黄金法则

> 「问鼎（品牌防守） + 顶展（精准拦截） + P4P（主动进攻） + 揽月（品牌延伸）」四位一体

**排列规律（搜索结果页）**:
```
位置 1: 问鼎
位置 2: 顶展
位置 3: P4P / 自然排名（优爆品橱窗位）/ 揽月 交织
```

**行动点**: 做组合营销，确保搜索结果页你的品牌/产品至少出现 3 次以上。

## 六、P4P 投流完整方法论

### 6.1 P4P 投流前 — 产品质量审查

> **核心原则**: 广告是「放大优势」而非「掩盖短板」。产品必须已有「高转化基因」才投放。

投前检查清单:
- [ ] 标题是否符合 128 字符/关键词覆盖
- [ ] 主图是否为场景化展示（告别单调白底图）
- [ ] 商品卖点是否充分利用 2000 字符
- [ ] 产品是否至少达到「潜力优品」等级
- [ ] 价格是否在第一页同行区间内
- [ ] 是否有认证（酒类需 FDA/清真等）

### 6.2 P4P 三大计划架构

| 计划类型 | 预算占比 | 关键词策略 | 匹配方式 | 目标 |
|---------|---------|-----------|---------|------|
| **核心产品计划** | 70% | 高转化长尾词 + 四星词 | **精确匹配** | 稳定 ROI |
| **测款拉新计划** | 30% | 测试潜力品市场反应 | 广泛匹配 + 否定词 | 发现新增量 |
| **定向攻击计划** | 额外 | 竞品关键词 | 精确匹配 | 精准截流 |

### 6.3 选词逻辑

| 词类 | 特征 | 策略 |
|------|------|------|
| 四星词 | 高相关 + 中等竞争 | **优先选择，主打** |
| 长尾蓝海词 | 搜索明确、竞争度低、转化高 | 分组管理、单独出价 |
| 高转化词 | 历史有询盘/成交 | 提高出价，重点保护 |
| 热门大词 | 高竞争、高花费 | 新店避开，成熟期选择性投放 |
| 竞品词 | 同行在投的核心词 | 定向攻击计划中使用 |

### 6.4 出价与调优策略

#### 出价技巧

| 时段 | 操作 | 原因 |
|------|------|------|
| 欧美上午 9-11 点 | 提高出价 10-20% | 目标买家活跃时段 |
| 非核心时段 | 降低出价或暂停 | 避免无效消耗 |
| 每天 | 屏蔽无效点击 | 关注搜索词报告 |

#### P4P 关键词管理动作表

| 关键词表现 | 操作 | 频率 |
|-----------|------|------|
| 高花费无询盘 | → 加入否定列表 | 每周 |
| 低花费有询盘 | → 提高出价 | 每周 |
| 有曝光低点击 | → 优化主图+标题 | 每两周 |
| 高点击低转化 | → 优化详情页+交易条件 | 每两周 |
| 零曝光 | → 检查出价/产品或替换 | 每日 |

### 6.5 全站推（智能推广）

| 产品类型 | 推荐计划 | 定向标签 |
|---------|---------|---------|
| 新品 | 新品加速 | 无特殊定向 |
| 潜力品 | 潜力品成长 | 交易类: 潜在交易买家、高回访买家 |
| 优品 | 优品助推 | 定制类: 定制偏好、中大采购规模买家 |
| 爆品 | 优品助推（单独计划） | L3+ 买家溢价 |

**全站推核心设置**:
1. 选对主推地域（关闭低效国家）
2. 精准人群标签
3. 根据数据实时调整 — 点击高无询盘立即暂停，转化低优化详情页

### 6.6 L3+ 买家溢价

| 功能 | 说明 |
|------|------|
| L3+ 定义 | 高活跃、高采购量的买家标签 |
| 溢价机制 | 对 L3+ 买家提高出价百分比 |
| 适用场景 | 核心产品计划 + 全站推 |
| 效果 | 提高高质量商机比例，辅助星等级加分 |

### 6.7 搜索词降权与自诊断

**搜索词降权**:
- 平台定期对低效关键词降权
- 需关注官方通知

**全站推自诊断指南**:
- 平台内置诊断工具，分析：
  - 曝光不足原因（出价低/关键词少/产品少）
  - 点击不足原因（主图/价格/评价）
  - 询盘不足原因（详情页/交易条件/认证）
- 每周应运行一次诊断

## 九、数据分析 SOP

| 表现 | 诊断 | 行动 |
|------|------|------|
| 曝光高但点击低 | 主图不吸引 / 标题不够精准 | 改主图 + 优化标题 |

## 十、实战避坑指南

| # | 坑 | 后果 | 正确做法 |
|---|-----|------|---------|
| 1 | 关键词堆砌（重复同一词） | 被降权 | 每个想法最多重复 1-2 次 |

| 5 | 忽略售后 | 店铺评分下降 | 24 小时内响应客诉 |
| 6 | 橱窗位放普通品 | 浪费排名加成 | 优爆品橱窗占比 100% |
| 7 | P4P 投放低质品/普通品 | 烧钱无转化 | 仅投放潜力优品及以上 |

## 十一、分层运营成长路径

```
Phase 1: 冷启动 (出口通)
  → 低价引流 + 智能发品 + 基础 P4P 测品
  → 目标: 潜力优品占比 > 30%

Phase 2: 加速期 (出口通 / 准备升级金品)
  → 行业定向征品 + 广告组合 + 橱窗优化
  → 目标: 优品占比 > 50%, 达到二星

Phase 3: 爆发期 (金品诚企)
  → 问鼎/顶展 + P4P 多计划 + L3+ 溢价
  → 目标: 爆品打造, 三星以上

Phase 4: 统治期 (金品 Pro / 行业领袖)
  → 品牌推广 + 揽月 + 全站推稳健投放
  → 目标: 核心词第一页, 四星以上
```
