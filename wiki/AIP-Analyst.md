# AIP Analyst

## 定义

**AIP Analyst** 是 Palantir AIP 中的代理式工作流界面，使用自然语言对 [[Ontology]] 进行临时分析。AI 自主搜索 Ontology、创建对象集、转换数据、生成摘要和可视化。

## 核心属性

### 可用工具类别

| 工具类别 | 功能 |
|---------|------|
| 发现工具 | 发现相关的 Object Type 和 Dataset |
| 分析工具 | 探索和分析检索到的数据 |
| 操作工具 | 执行 Foundry 函数和 Ontology Action |
| 可视化工具 | Vega 图表、地图可视化、Mermaid 图表 |
| 文件处理 | 上传 Excel/CSV/Word/PDF/图像 |

### 注意事项

- 关闭会话后对话历史不保留，需手动保存查询和 Vega-Lite 代码
- 可导出分析会话为 PDF
