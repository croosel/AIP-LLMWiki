# 经验记录模板

> 本页面为 Agent 经验记录的标准模板。每次任务完成后，Agent 应按此格式记录经验。
> 新记录应创建为独立页面，文件名格式：`EXP-{简述}-{日期}.md`

---

以下为模板正文（Agent 创建新经验记录时复制以下内容）：

## 模板开始

```yaml
---
tags:
  - 经验记录
  - {领域标签: #Palantir / #LILIS / #外贸跨境 / #AIP}
  - {结果标签: #成功 / #失败 / #部分成功}
created: {YYYY-MM-DD}
agent: {执行任务的 Agent 名称}
task_id: {任务编号，如无则留空}
outcome: {success / failure / partial}
confidence: {high / medium / low}
related_wiki: 
  - [[相关Wiki页面1]]
  - [[相关Wiki页面2]]
---
```

# {任务简述}

- **时间**: {执行时间}

- **选择的方法**: {用了什么方法，为什么选这个方法}
