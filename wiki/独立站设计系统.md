# LILIS 独立站设计系统

> 本文档定义 LILISdecor.com 独立站的完整视觉规范，确保品牌一致性。
> 品牌基础参考 [[LILIS品牌手册]]，Logo 概念参考 [[AIP-Beacon-产品命名与品牌定位]]。

---

## 一、品牌关键词

```
Professional × Warm × Celebration-Driven × Trust-Oriented

翻译为设计语言:
  Professional   → 留白充足、排版克制、信息层级清晰
  Warm           → 暖色系点缀、圆角元素、人文场景图
  Celebration    → 色彩饱和度适中、动感构图、节日氛围
  Trust-Oriented → 规范图标、认证徽章、数据展示
```

---

## 二、配色体系

### 主色板

| 色名 | HEX | RGB | 用途 |
|------|-----|-----|------|
| **Sage Green** | `#7C9070` | 124,144,112 | 主品牌色 — 呼应婚庆花艺的自然感 |
| **Warm Ivory** | `#FAF6F0` | 250,246,240 | 页面背景色 — 温暖、不刺眼 |
| **Deep Charcoal** | `#2D2D2D` | 45,45,45 | 主文字色 — 比纯黑柔和 |
| **Soft Gold** | `#C8A96E` | 200,169,110 | 强调色 — 呼应奢华/庆典感 |

### 辅色板

| 色名 | HEX | 用途 |
|------|-----|------|
| **Blush Pink** | `#F2D7D5` | 浪漫品类/婚庆主题点缀 |
| **Dusty Rose** | `#C9A9A6` | 次强调色/价格标签 |
| **Sage Light** | `#E8EDE5` | 卡片背景/分区背景 |
| **Charcoal Light** | `#6B6B6B` | 次文字/描述文字 |
| **White** | `#FFFFFF` | 卡片/弹窗/模态框 |
| **Error Red** | `#D4453A` | 表单错误/库存告警 |
| **Success Green** | `#4A8C5C` | 成功状态/库存有货 |

### 配色规则

```
1. 每个页面不超过 3 种主色 (Sage Green + Warm Ivory + 1 强调色)
2. Soft Gold 仅用于 CTA 按钮 / 价格 / 徽章
3. Blush Pink 仅用于婚庆品类页面和情人节/婚礼季专题
4. 文字对比度: Deep Charcoal on Warm Ivory ≥ 7:1 (WCAG AAA)
5. 次文字对比度: Charcoal Light on Warm Ivory ≥ 4.5:1 (WCAG AA)
```

## 三、字体系统

### 主字体

| 用途 | 字体 | 回退 | 理由 |
|------|------|------|------|
| **标题 H1-H3** | Playfair Display | Georgia, serif | 衬线体传达优雅/经典婚庆感 |
| **正文/按钮** | Inter | -apple-system, sans-serif | 无衬线体保证可读性 |
| **价格/数据** | Inter (tabular nums) | — | 等宽数字对齐 |

### 字号阶梯

| 元素 | 桌面端 | 移动端 | 字重 | 行高 |
|------|--------|--------|------|------|
| H1 (页面标题) | 48px | 32px | 700 (Playfair) | 1.2 |
| H2 (区块标题) | 36px | 24px | 600 (Playfair) | 1.3 |
| H3 (卡片标题) | 24px | 20px | 600 (Playfair) | 1.3 |
| Body (正文) | 16px | 15px | 400 (Inter) | 1.6 |
| Small (描述) | 14px | 13px | 400 (Inter) | 1.5 |
| Caption (标签) | 12px | 12px | 500 (Inter) | 1.4 |
| Button (按钮) | 16px | 15px | 600 (Inter) | 1.0 |

### 字体规则

```
1. 标题绝不全部大写（仅品牌名 LILIS 全大写）
2. 产品名用 Title Case: "Silk Peony Bridal Bouquet"
3. 正文段宽 ≤ 70 字符/行
4. 中文内容（如有）用 Noto Sans SC，不用系统宋体
```

## 五、组件库

### 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| **Primary** | 背景 Soft Gold `#C8A96E`，文字 White，圆角 4px | 主 CTA (加购/询盘/订阅) |
| **Secondary** | 边框 Sage Green `#7C9070`，文字 Sage Green，圆角 4px | 次 CTA (查看详情/浏览更多) |
| **Text** | 文字 Sage Green，无背景无边框 | 链接/导航 |
| **Disabled** | 背景 `#E0E0E0`，文字 `#999` | 禁用状态 |

按钮尺寸:

| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| Large | 52px | 16px 32px | 16px |
| Default | 44px | 12px 24px | 15px |
| Small | 36px | 8px 16px | 13px |

### 产品卡片

```
┌──────────────────────┐
│                      │
│    [产品图片 1:1]      │  ← 正方形，object-fit: cover
│                      │
├──────────────────────┤
│ 品类标签 (caption)     │  ← Sage Green 文字
│ 产品名 (H3)           │  ← 2 行截断
│ ★★★★☆ (24 reviews)   │  ← Soft Gold 星星
│ From $3.50/piece      │  ← Soft Gold 价格
│ MOQ: 50 pcs           │  ← Charcoal Light
│ [Get Quote]           │  ← Secondary 按钮
└──────────────────────┘
```

卡片间距: 桌面 24px，移动 16px
卡片圆角: 8px
卡片阴影: `0 2px 8px rgba(0,0,0,0.06)` hover 时 `0 4px 16px rgba(0,0,0,0.1)`

### 信任徽章

| 徽章 | 图标 | 文字 |
|------|------|------|
| US Warehouse | 🏠 | US Warehouse · 3-5 Day Delivery |
| Trade Assurance | 🛡️ | Alibaba Trade Assurance Verified |

尺寸: 图标 24×24 + 文字 14px，横排排列，背景 White + 圆角 4px + 浅边框

### 表单

| 元素 | 样式 |
|------|------|
| 输入框 | 高度 44px，边框 1px `#D0D0D0`，focus 时边框 Sage Green |

## 六、图片规范

### 产品图

| 类型 | 尺寸 | 比例 | 背景 |
|------|------|------|------|
| 主图 | 1200×1200px | 1:1 | 白色 `#FFFFFF` |
| 场景图 | 1200×800px | 3:2 | 真实场景 |
| 细节图 | 800×800px | 1:1 | 白色或中性背景 |
| 尺寸参照图 | 800×800px | 1:1 | 含标尺/手模 |

### 场景图拍摄指南

```
1. 每个产品至少 2 张场景图（婚礼现场/派对布置）
2. 灯光: 自然光优先，避免过曝
3. 道具: 真花/真布艺搭配，体现品质感
4. 色温: 暖调 (5200K-5600K)
5. 构图: 产品占画面 40-60%，留出环境氛围
6. 风格统一: 所有场景图同后期色调
```

### Banner/Hero 图

| 位置 | 尺寸 | 比例 |
|------|------|------|
| 首页 Hero (桌面) | 1920×800px | 2.4:1 |
| 首页 Hero (移动) | 800×1000px | 4:5 |
| 品类页 Banner | 1200×400px | 3:1 |
| 博客封面 | 1200×630px | 1.9:1 (OG 标准) |

## 七、动效与交互

| 交互 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 按钮 hover | 背景色加深 10% | 150ms | ease |
| 卡片 hover | 阴影增大 + 微上移 2px | 200ms | ease-out |
| 图片 hover | 轻微放大 1.03x | 300ms | ease |
| 页面切换 | 淡入 | 200ms | ease |
| 弹窗出现 | 从下方滑入 + 背景遮罩 | 250ms | ease-out |
| Toast 通知 | 从顶部滑入 | 200ms | ease-out |

**原则**: 动效服务于反馈，不装饰。任何动效不超过 300ms。

## 八、响应式断点

| 断点 | 宽度 | 导航 | 卡片列数 | 侧边栏 |
|------|------|------|---------|--------|
| Mobile S | ≤ 375px | 汉堡菜单 | 1 | 隐藏 |
| Mobile | ≤ 768px | 汉堡菜单 | 2 (小卡片) | 隐藏 |
| Tablet | 769-1024px | 折叠菜单 | 2-3 | 可选 |
| Desktop | ≥ 1025px | 完整导航 | 3-4 | 显示 |

**移动端优先原则**: 60%+ 婚庆搜索来自移动端，移动端不是桌面端的缩放版。
