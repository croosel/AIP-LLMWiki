# 信保 (Trade Assurance)

## 定义

**信保 (Trade Assurance)** 是 Alibaba 的担保交易服务，保障买卖双方的交易安全。信保订单的成交额计入 [[商家星等级|交易分]]，且信保成交价有硬性底价约束（不得低于页面价的 50%）。

## 核心属性

### 信保底价规则

| 规则 | 后果 |
|------|------|
| 信保订单实际成交价 ≥ 页面价的 50% | 正常计入销量和权重 |
| 信保订单实际成交价 < 页面价的 50% | **成交不计销量**，权重受损 |
| 成交不卖 | 价格权重受损 |

### 相关概念

| 概念 | 说明 |
|------|------|
| ACP | Alibaba 的担保交易模式之一 |
| ACP+ | ACP 的升级版，提供更高保障 |
| 180 天最低价 | 页面价、折扣价、成交价三者取最低 |

## 业务用途

- 积累[[商家星等级|交易分]]（快速升星的核心指标）
- 买家信任背书（有信保标识的产品转化率更高）
- 与 [[P4P投流完整方法论|P4P]] 协同，信保成交越多 P4P 权重越高

## 外贸应用

> 参见 [[LILIS-婚庆派对用品]]

### LILIS 信保现状

- 近 60 天交易额: **$0 USD**
- 交易力: **0 分**
- 根因: 开店约 1 年但无信保订单积累

> **行动**: 通过老客邀约和低价引流产生首批信保订单，打破零交易的局面。

## 常见问题

**Q: 信保和线下交易有什么区别？**
A: 信保走 Alibaba 平台担保，成交额计入星等级交易分；线下交易不计入。

**Q: 样品单能不能走信保？**
A: 可以，信保支持小额定单。样品单也是积累交易分的有效方式。
