# 商家星等级

## 定义

**商家星等级 (Star Rating)** 是 Alibaba 国际站对卖家综合能力的评级体系（0-5 星）。星等级**直接影响流量的上限而非下限**——同等条件下，高星店铺可获得更多曝光分配。

## 核心属性

### 评分构成

| 分项 | 核心指标 | 提升方式 |
|------|---------|---------|
| **商品力** | 全店优品数、橱窗优品占比、潜力优品/优品/爆品占比 | 提升优品数量 + [[橱窗策略|橱窗优品占比 100%]] |
| **营销力** | 营销流量指数、商机指数、商机转化率 | [[P4P投流完整方法论|P4P 投流]] + 全站推 |
| **交易力** | 近 60 天站内/站外交易额 | 信保订单 + ACP + ACP+ |
| **服务力** | 平均回复时间 | 保持 ≤ 2h 回复 |

### 商机分 & 交易分

| 分数 | 说明 |
|------|------|
| **商机分** | 商机数量 + 商机质量。小满获取的商机有额外加分。通过 P4P 获取的高质量商机也有额外加分 |
| **交易分** | 信保订单 + ACP + ACP+。通过 Accio Work 分析买家背景可快速提高成交额 |

### 星级保效（付费直达）

| 星级 | L+ 买家浓度保障 | 适用阶段 |
|------|---------------|---------|
| 三星直达 | L2+ 买家浓度 ≥ 25% | 起步/成长 |
| 四星直达 | L2+ 买家浓度 ≥ 30% | 发展 |
| 五星直达 | L3+ 买家浓度 ≥ 20% | 行业头部 |

## 业务用途

- 直接影响流量分配和买家信任
- 0 星店铺几乎没有自然流量，必须靠 P4P 补充
- 高星店铺可获得更多活动和展馆露出

## 常见问题

**Q: 星等级多久更新一次？**
A: 每月更新，基于近 90 天数据滚动计算。

**Q: L2+ / L3+ 买家是什么意思？**
A: L2+ 为中等活跃采购买家，L3+ 为高活跃高采购量买家。星级保效的 L+ 浓度代表该类买家在你的流量中的占比。

**Q: 从 0 星升到 1 星需要多久？**
A: 取决于能否快速产生信保订单。启动 P4P + 释放 23 个优品入橱窗，预计 30-60 天可达 1 星。
