# 转化率优化体系

## 定义

Alibaba 国际站的转化率优化（CRO）覆盖从 **搜索曝光 → 点击 → 询盘 → 成交** 的全链路。核心公式：

```
曝光量 × 点击率 × 商机转化率 × 成交率 = 销售额
```

每个环节的优化都是独立且可操作的。广告放大曝光，但转化率决定钱烧得值不值。

## 核心属性

### 漏斗诊断模型

| 表现 | 诊断 | 行动 |
|------|------|------|
| 曝光高但点击低 | 主图不吸引 / 标题不够精准 | 改主图 + 优化[[标题优化黄金公式|标题]] |
| 点击高但询盘少 | 详情页痛点解决方案不足 | 优化前 3 屏 + 添加客户案例 |
| 询盘多但成交少 | 价格/[[MOQ]]/交期不符合预期 | 调整交易条件 |
| 零曝光 | 关键词不匹配/价格过高 | 换关键词或降价冲[[信保]] |

### 视觉呈现标准

| 元素 | 标准 | 黄金法则 |
|------|------|---------|
| 主图 | 场景化展示 | 避免单调白底图 |
| 前 3 屏 | 解决客户痛点 | 如质量担忧 → 放 QC 流程图 |
| 产品参数 | 表格对比 | 一行一个参数 |
| 客户案例 | 真人场景图 + 简短评价 | 真实感 |

### RFQ 报价策略

- 优先选择「标星」买家（历史采购记录多）
- 需求描述清晰的 RFQ
- 报价附：产品链接 + 3 张实拍图 + [[MOQ]] + 交货期

## 业务用途

- 放大广告 ROI（同预算下多出 30-50% 询盘）
- 通过详情页优化降低跳出率
- RFQ 报价质量直接影响成交概率

## 常见问题

**Q: 主图用白底图还是场景图？**
A: 主图必须场景化。白底图只能做辅图。场景图（如婚礼拱门实景）是点击率的第一驱动力。

**Q: 详情页前 3 屏应该放什么？**
A: 第 1 屏核心卖点（如 Real Touch 工艺），第 2 屏解决质量担忧（QC/认证），第 3 屏应用场景展示。
