# AIP Assist

## 定义

**AIP Assist** 是 Palantir AIP 中的 LLM 驱动支持工具，帮助用户导航 Palantir 平台。它以侧边栏形式嵌入所有平台应用，提供上下文感知的实时自然语言辅助。

## 核心属性

- 键盘快捷键: `Cmd+Shift+U` (macOS) / `Ctrl+Shift+U` (Windows)
- 可从任意平台应用侧边栏打开
- 根据活跃应用不同，回答内容也不同
- 支持多语言
- 可注册自定义文档类型仓库（通过 Code Repositories）
- 两种增强方式：添加到默认知识库 或 创建 AIP Assist Chatbots

## 常见问题

**Q: AIP Assist 和 AIP Chatbot 的区别？**
A: Assist 是平台级帮助工具（回答"怎么用"），Chatbot 是业务级交互助手（执行"做什么"）。
