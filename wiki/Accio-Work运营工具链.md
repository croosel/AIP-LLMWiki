# Accio Work 运营工具链

## 定义

**Accio Work** 是 AI 驱动的运营工具平台，为 Alibaba 国际站卖家提供智能发品、数据分析、同行统计、买家分析和店铺诊断服务。与官方插件（生意助手、Maxwell、数据参谋）形成互补工具链。

## 核心属性

### 五大应用场景

| 场景 | Accio Work 角色 | 产出 |
|------|----------------|------|
| **智能发品** | 关键词整理 + 市场分析 + [[标题优化黄金公式|标题]]生成 + 卖点撰写 + 主图生成 | 完整产品 Listing |
| **数据分析** | 上传产品参谋周/月报表，自动诊断 | Excel 高级排版报表 |
| **同行统计** | 统计关键/长尾词第一页销量 | 竞品分析报告 |
| **买家[[询盘]]分析** | 分析买家背景辅助成交决策 | 买家画像 + 建议回复 |
| **店铺诊断** | 对标 SOP 规则，输出优化清单 | 店铺健康报告 |

### 官方插件生态

| 插件 | 功能 |
|------|------|
| 国际站生意助手 | 智能发品、店铺分析 |
| 商品运营助手 Maxwell | 优爆商品打造 / 优品保险计划 |
| 数据参谋-产品参谋 | 产品数据周/月筛选导出 |

### 建议 Prompt 模板

```
"这是我的阿里巴巴国际站店铺上周的整体数据，请为我分析其中
高曝光低点击和高点击低询盘和TM转化的商品，罗列前20名，
并给我具体的优化建议，以Excel表格的形式输出给我，要求高级排版"
```

## 业务用途

- 替代手动发品流程（关键词整理→标题→卖点→主图）
- 将店铺诊断自动化
- 辅助[[商家星等级|交易分]]提升（买家背景分析→快速成交）

### 分层运营成长路径

```
Phase 1: 冷启动 (出口通)
  → 低价引流 + 智能发品 + 基础 P4P 测品
  → 目标: 潜力优品占比 > 30%

Phase 2: 加速期
  → 行业定向征品 + 广告组合 + 橱窗优化
  → 目标: 优品占比 > 50%, 达到二星

Phase 3: 爆发期 (金品诚企)
  → 问鼎/顶展 + P4P 多计划 + L3+ 溢价
  → 目标: 爆品打造, 三星以上

Phase 4: 统治期 (金品 Pro)
  → 品牌推广 + 揽月 + 全站推
  → 目标: 核心词第一页, 四星以上
```
