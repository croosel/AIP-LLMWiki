# 商品卖点 (Selling Points)

## 定义

**商品卖点** 是 Alibaba 国际站产品详情页的独立文本字段，限 2000 字符。原关键词和副标题已合并至此，**该部分全部进入搜索索引**——是 AI Mode 下最被低估的 SEO 金矿。

## 核心属性

| 属性 | 说明 |
|------|------|
| 字符上限 | 2000 字符 |
| 搜索权重 | 全部进入搜索索引 |
| 历史变化 | 原关键词和副标题已合并到此字段 |
| 内容建议 | 产品核心卖点、应用场景、认证信息、FAQ |

### 与标题的关系

| 字段 | 字符 | 搜索权重 | 定位 |
|------|------|---------|------|
| [[标题优化黄金公式|标题]] | 128 | 匹配第一要素 | 吸引点击 |
| 商品卖点 | 2000 | 语义匹配补充 | 承载详细信息 |

## 业务用途

- 扩展搜索覆盖（128 字符标题不够，2000 字符卖点补齐）
- 提升 AI Mode 下的语义匹配率
- 集中展示认证、场景、材质等决策信息

## 外贸应用

> 参见 [[Alibaba排序逻辑#隐性权重 — 相关性匹配模型]]

### 卖点撰写建议

```
第 1 段: 产品核心优势 (Material / Craft / Certification)
第 2 段: 应用场景 (Wedding / Party / Hotel / Event)
第 3 段: 服务承诺 (MOQ / Delivery / Customization)
第 4 段: FAQ (高频问题预设答案)
```
