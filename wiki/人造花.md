# 人造花 (Artificial Flower)

## 定义

**人造花 (Artificial Flower)** 是 LILIS 品牌的主营品类，指使用丝绸（Silk）、涤纶（[[涤纶纤维|Polyester]]）、乳胶（Latex）等材料仿制的花卉装饰品。与真花相比，具有可重复使用、不受季节限制、颜色稳定等优势。

## 核心属性

| 属性 | 说明 |
|------|------|
| 材质 | Silk（丝绸感）、Polyester（涤纶，成本低）、Latex（乳胶，高仿真） |
| 尺寸 | 20cm 花头、7-Fork（七叉分枝）等规格 |
| 用途 | Wedding（婚礼）、Party（派对）、Home Decor（家居）、Garden（花园） |
| Real Touch | 高端仿真技术，触感和外观接近真花 |
| Customizable | 支持颜色、尺寸、组合定制 |

## 业务用途

- 婚礼拱门花艺装饰
- 花墙背景搭建
- 桌花中心装饰（Centerpiece）
- 酒店大堂装饰

## 外贸应用

- 旺季（5-10 月）是主要出货品类

- 搭配链: 人造花 → 拱门支架 → 灯光霓虹 → 桌布

## 常见问题

**Q: Real Touch 和普通 Silk Flower 有什么区别？**
A: Real Touch 使用聚氨酯（PU）涂层工艺，触感柔弹、花瓣有纹理，视觉仿真度远高于普通丝绸花。价格通常高 2-3 倍。

**Q: 人造花在欧美市场的合规要求？**
A: 需符合 REACH/RoHS 化学安全标准，特别是染色剂和阻燃剂的含量限制。
