# 橱窗策略

## 定义

**橱窗 (Showcase Window)** 是 Alibaba 国际站的付费展示位。在同等条件下，**橱窗商品的排名必定靠前**。橱窗位是最确定性的排名提升手段。

## 核心属性

### 三大铁则

| 原则 | 说明 |
|------|------|
| 天然排名优势 | 同等条件下橱窗商品排名必定靠前 |
| **优爆品橱窗占比 100%** | 橱窗位只放优品和爆品，绝不允许普通品占位 |
| 动态替换 | 按周/月根据[[询盘|商机]]数据替换橱窗商品 |
| 位置排序 | 数据好的橱窗商品放在靠前位置（营销中心 → 橱窗 → 排序） |

### 操作 SOP

```
Step 1: 产品参谋 → 导出商品分层报表
Step 2: 勾选所有优品和爆品
Step 3: 按商机数据（询盘 + TM）降序排列
Step 4: 前 N 个商品放入橱窗（N = 橱窗位数量）
Step 5: 营销中心 → 橱窗 → 按数据排序
Step 6: 每周复盘：数据差的替换为新优品
```

## 业务用途

- 最快最确定性的排名提升手段
- 让优品获得"确定性的第一页位置"
- 与 [[P4P投流完整方法论|P4P]] 协同实现"自然 + 付费"双曝光

## 外贸应用

- **行动**: 立即将 23 个优品全部入橱窗，实现 P0 级优化
- **预期效果**: 优品曝光 +55%（2901→4500/周）

## 常见问题

**Q: 橱窗位有限怎么办？**
A: 按商机数据（询盘+TM）降序排列，优先保留转化数据最好的商品。每周复盘替换低效商品。

**Q: 橱窗商品可以同时投 P4P 吗？**
A: 可以且推荐。橱窗给自然排名优势，P4P 给付费排名补充，搜索结果页可实现"双坑位曝光"。
