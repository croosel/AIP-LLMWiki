## 四、关键交互流程

### 4.1 询盘流程 (B2B 核心路径)

```
用户点击 "Get Quote"
  → 跳转 /pages/contact 或弹窗表单
    → 填写: 姓名/邮箱/公司/产品/MOQ/消息
      → 提交
        → 1) 即时确认页 "We'll respond within 3 hours"
        → 2) 自动邮件 (Klaviyo 触发)
        → 3) 人工跟进 ≤3h
        → 4) 后续培育流 (3天/7天/14天 自动邮件)
```

### 4.2 购物车流程 (B2C 路径)

```
加购 → Cart 页面
  → 修改数量/删除
    → 进入 Checkout
      → 填写地址/支付
        → 订单确认页
          → 感谢邮件 + 物流跟踪
```

### 4.3 Email 订阅流程

```
输入邮箱 → 点击 Subscribe
  → 确认弹窗 "Check your inbox"
    → Welcome 邮件 (含 10% 首单折扣码)
      → 培育序列:
          Day 1: Welcome + 品牌故事
          Day 3: 热门产品推荐
          Day 7: 婚礼策划指南
          Day 14: 批发专属优惠
          Monthly: 新品/季节内容
```

### 4.4 废弃购物车挽回

```
加购未结账 → 离站
  → 1h 后: Klaviyo 邮件 "Still thinking? Your cart is waiting."
  → 24h 后: 邮件 + 5% 折扣码 (如仍未回)
  → 72h 后: Retargeting 广告 (Facebook/Instagram)
```
