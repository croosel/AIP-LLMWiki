# 全站推与智能推广

## 定义

**全站推 (Site-wide Promotion)** 是 Alibaba 国际站的智能推广产品，根据产品等级自动匹配推广策略。与 [[P4P投流完整方法论|P4P]] 的关键词手动管理不同，全站推由平台算法自动选词和出价。

## 核心属性

### 按产品等级匹配

| 产品类型 | 推荐计划 | 定向标签 |
|---------|---------|---------|
| 新品 | 新品加速 | 无特殊定向 |
| 潜力品 | 潜力品成长 | 交易类: 潜在交易买家、高回访买家 |
| 优品 | 优品助推 | 定制类: 定制偏好、中大采购规模买家 |
| 爆品 | 优品助推（单独计划） | L3+ 买家溢价 |

### L3+ 买家溢价

| 维度 | 说明 |
|------|------|
| L3+ 定义 | 高活跃、高采购量的买家标签 |
| 溢价机制 | 对 L3+ 买家提高出价百分比 |
| 适用场景 | 核心产品计划 + 全站推 |
| 效果 | 提高高质量商机比例，辅助 [[商家星等级|星等级]] 加分 |

### 核心设置三要素

1. **选对主推地域** — 关闭低效国家
2. **精准人群标签** — 交易类 / 定制类定向
3. **实时数据调整** — 点击高无询盘立即暂停，转化低优化详情页

## 业务用途

- 替代部分 P4P 的手动关键词管理，降低运营复杂度
- 通过 L3+ 溢价锁定高质量买家
- 潜力品成长的自动化引擎

## 外贸应用

### 全站推自诊断指南

平台内置诊断工具，每周运行一次，分析：

| 问题 | 可能原因 |
|------|---------|
| 曝光不足 | 出价低 / 关键词少 / 产品少 |
| 点击不足 | 主图 / 价格 / 评价问题 |
| 询盘不足 | 详情页 / 交易条件 / 认证缺失 |

## 常见问题

**Q: P4P 和全站推可以同时使用吗？**
A: 可以。P4P 手动管理核心词，全站推覆盖长尾和测款。两者互补而非重复。

**Q: L3+ 溢价应该设置多高？**
A: 建议从 20% 开始，观察 L3+ 询盘的转化质量后逐步调整。
