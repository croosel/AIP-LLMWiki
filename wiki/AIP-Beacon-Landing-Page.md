## 常见问题

**Q: AIP Beacon 是独立 App 吗？**
A: 不是。它是一套预配置的 Agent 包，运行在 Accio Work 平台上。你需要一个 Accio Work Pro 账号（$20/月，由 Accio Work 收取）。

**Q: 数据安全吗？**
A: Beacon 直接从你的 Alibaba 店铺后台读取数据（通过 MCP 连接），数据存储在 Accio Work 的隔离空间内。我们看不到你的店铺数据——只有你自己的 Agent 能看到。

**Q: 不想用了怎么办？**
A: 随时取消。数据保留 30 天。没有锁定期，没有取消费。

**Q: 支持哪些平台？**
A: 当前阶段：Alibaba.com（国际站）。未来：UCP 协议（覆盖 Shopify、Google Shopping）和 Amazon。

**Q: 如果我是一个代理运营（代运营），管 5 个店铺？**
A: 选 Custom 版（$299/月），支持多店铺 + 定制规则。
